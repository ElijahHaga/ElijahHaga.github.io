LED Ring Circuit Board

This circuit board is an LED ring PCB that I designed in Flux, a free browser-based ECAD. It uses an STM32F030CCT6 microcontroller and a BMI160 6-axis gyroscope to control 20 LEDs depending on the orientation of the circuit board. It is charged an programmed by a USB-C port. I thought of, researched, and designed this circuit board completely on my own, doing my own parts selection and BOM while learning how to use Flux. I definitely prefer Altium Designer, but I used Flux to see how well I could adapt my skills to a completely different ECAD.

Raspberry Pi Pico Clone

This circuit board is a clone of a Raspberry Pi Pico that I deisgned to improve my ECAD skills. I used Altium Designer and the Celestial Altium Library, a free online library of components for this circuit board, although I designed the RP2040 model and a few other components myself. The circuit board uses an RP2040 microcontroller, which can be programmed using the Raspberry Pi operating system and Python. This was my first project in computer/control boards, which is the board design I like most.
