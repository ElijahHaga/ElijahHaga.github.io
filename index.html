Electrical Engineering Portfolio

LED Ring Circuit Board

This circuit board is an LED ring PCB that I designed in Flux, a free browser-based ECAD. It uses an STM32F030CCT6 microcontroller and a BMI160 6-axis gyroscope to control 20 LEDs depending on the orientation of the circuit board. It is charged an programmed by a USB-C port. I thought of, researched, and designed this circuit board completely on my own, doing my own parts selection and BOM while learning how to use Flux. I definitely prefer Altium Designer, but I used Flux to see how well I could adapt my skills to a completely different ECAD.

Raspberry Pi Pico Clone

This circuit board is a clone of a Raspberry Pi Pico that I deisgned to improve my ECAD skills. I used Altium Designer and the Celestial Altium Library, a free online library of components for the simplier electronics of circuit board, although I designed the RP2040 model and a few other more complicated components myself. The circuit board uses an RP2040 microcontroller, which can be programmed using the Raspberry Pi operating system and Python. This was my first project in computer/control boards, which is the board design I like most.

Gameboy Color Cartridge

This PCB is a Gameboy Color Cartridge circuit board that I reverse engineered. I researched the parts in the original cartridges, recreated them in modern 3D models and footprints, and arranged them in the same positions they are in in the original cartridges, but I routed the traces and placed the copper pour to my own design. It uses an MBC5 chip to map storage to a MX23C8005-12 read-only memory (ROM) to store the game files, and a W24257S-70LL random access memory (RAM) chip to store the save states for the player. It also has a MM1134A as a back-up IC. It interfaces with the Gameboy Color using a 32-pin card edge slot. I used Altium Designer and the Celestial Altium Library for this project as well. This was another project that I did to improve my ECAD skills and to practice reverse engineering of circuit boards.
